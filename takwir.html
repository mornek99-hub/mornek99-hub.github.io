AYAT 01 إِذَا الشَّمْسُ كُوِّرَتْ إذا الشمس كورت "Wenn das Display/die Energiequelle (Shams) komprimiert/gezippt wird (Kuwwirat)."
K-W-R (Compress/Zip): Das Einfalten der Dimensionen. Komprimierung von Daten.
Sh-M-S (Source/Display): Die visuelle Energiequelle wird abgeschaltet.
AYAT 02 وَإِذَا النُّجُومُ انكَدَرَتْ وإذا النجوم انكدرت "Und wenn die Knotenpunkte/Pixel (Nujum) das Signal verlieren (Inkadarat)."
N-J-M (Nodes): Netzwerk-Knoten oder Pixel auf dem Screen.
K-D-R (Signal Loss): Fade Out. Das Grid geht offline.
AYAT 07 وَإِذَا النُّفُوسُ زُوِّجَتْ وإذا النفوس زوجت "Und wenn die User-Profile (Nufus) gemerged/synchronisiert werden (Zuwwijat)."
N-F-S (User Profiles): Die Software-Agenten (Seelen).
Z-W-J (Merge): Zusammenführung von User-ID und Backup-Daten.
AYAT 10 وَإِذَا الصُّحُفُ نُشِرَتْ وإذا الصحف نشرت "Und wenn die Log-Files/Datenbanken (Suhuf) extrahiert/veröffentlicht werden (Nushirat)."
S-H-F (Logs): Die aufgezeichneten Datensätze.
N-Sh-R (Publish/Spread): Volle Transparenz. Keine verschlüsselten Ordner mehr.
AYAT 11 وَإِذَا السَّمَاءُ كُشِطَتْ وإذا السماء كشطت "Und wenn das Interface/Skybox (Sama) entfernt/gestripped wird (Kushitat)."
S-M-A (Interface): Die grafische Oberfläche (Himmel), die den Code verdeckt.
K-Sh-T (Strip/Remove): Das Entfernen der Skin. Der rohe Kernel wird sichtbar.
